Object 클래스
    - boolean equals(Object obj)
        객체 동등 비교 시 equals()사용
        객체의 주소값을 비교하고 boolean(true/false) 값을 리턴
    - int hashcode()
        객체 해시코드란 객체를 식별하는 정수를 의미함.
        Object 클래스의 hashcode() 객체의 메모리 번지(주소값)을 이용해서 해시코드를 생성한다.
        객체마다 다른 정수값을 리턴한다.

        hashcode()의 목적은 객체의 주소값을 기반으로 해싱해서 고유한 숫자값을 반환한다.
        객체가 같은지 같지 않은지 판별한다.
        만약에,객체 자체의 주소갑(해시값) 얻어야 하는 상황이라면 : java.util.Objects.identityHashCode()

    자바는 두 객체가 동등함을 비교할때, hashcode() + equals()메소드를 같이 사용하여 검증한다.
        hashcode() + equals()


    - String toString()

    - clone 메소드 : 객체를 깊은 복사(deep copy) 할 때 사용하는 메소드
    Object.clone()메소드는 인터턴스 객체의 복제를 위한 메소드

        ^ -> 시작의미. [^] 부정의 의미 ^a : a로 시작하는 단어 [^a] : a가 아닌 철자인 문자 1개
        $\ -> $앞의 문자열로 문자가 끝나는가? s$ : a로 끝나는 단어?
        [] -> [] 대괄호 안의 문자가 있는지를 확인 [ab][cd] : a,b중 한문자와 c,d중 한문자만 허용 ->ac ad bc bd만 입력가능
        [^] -> []안에 ^문자가 있으면 , 해당 문자 제외 전체 입력가능
            [^a-z]:알파벳 소문자 a부터 z까지를 제외한 문자 입력가능
